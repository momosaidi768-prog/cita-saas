# 🚀 Cita SaaS Bot

Telegram SaaS Bot for appointment tracking and user management with Stripe payments.

---

## 📌 Features

- 🤖 AI assistant (simple smart replies)
- 👤 User registration flow (name, NIE, city, email, phone)
- 💳 Stripe payment integration (9.99€)
- 🔔 Payment webhook (auto VIP activation)
- 🌐 Ready for deployment on Railway
- 📊 SQLite database for users

---

## ⚙️ Tech Stack

- Python 3
- Telegram Bot API (python-telegram-bot)
- Flask (webhooks)
- Stripe API
- SQLite database
